# Sistema de Gestión de Inventarios para PyMEs

Proyecto académico desarrollado en Java aplicando Programación Orientada a Objetos, con integración a base de datos SQL Server, orientado a mejorar la gestión de inventarios en pequeñas y medianas empresas.

## Tecnologías
- Java (Apache NetBeans)
- SQL Server
- Programación Orientada a Objetos (POO)
- Arquitectura por capas
- Procedimientos almacenados, funciones y triggers

## Funcionalidades
- Registro, edición y consulta de productos
- Control de entradas y salidas de inventario
- Alertas automáticas por bajo stock
- Sistema de autenticación con roles y permisos
- Generación de reportes con filtros (categoría, proveedor, stock, precios)

## Contexto
Proyecto académico realizado en el curso de Programación Orientada a Objetos.
